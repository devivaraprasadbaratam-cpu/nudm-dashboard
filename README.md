# NUDM Property Tax Analytics Dashboard

A modern analytics dashboard built for the NUDM Internship Technical Assessment using Next.js, TypeScript, Tailwind CSS, Recharts, and AI-powered analytics.

- KPI Dashboard
- Multi-city Filtering
- Property Tax Analytics
- Collection Comparison Chart
- AI Property Assistant
- Responsive Mobile UI
- JSON Dataset Integration

## 📦 Installation

Clone repository:

```bash
git clone https://github.com/devivaraprasadbaratam-cpu/nudm-dashboard.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create `.env.local`

```env
NEXT_PUBLIC_GEMINI_API_KEY=your_api_key
```
